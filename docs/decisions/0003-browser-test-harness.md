# 0003. Browser test harness

- 상태: 승인됨
- 날짜: 2026-07-13

## Context

성덕순례의 핵심 기능은 콘텐츠 탐색, 일정 생성, 지도 상호작용처럼 브라우저 동작에 의존한다.
정적 검사와 프로덕션 빌드만으로는 실제 페이지 실행과 사용자 흐름을 검증할 수 없다.

## Decision

- Playwright와 Chromium을 브라우저 검증 하네스로 사용한다.
- `pnpm verify`에 브라우저 테스트를 포함한다.
- 초기에는 앱 셸, 제품 메타데이터, 런타임 오류를 확인하는 스모크 테스트를 유지한다.
- 기능 구현과 함께 핵심 사용자 흐름을 같은 테스트 계층에 추가한다.

## Consequences

- 로컬 및 CI 완료 조건이 실제 브라우저 실행을 포함한다.
- 최초 로컬 실행에는 Playwright 브라우저 설치가 필요하다.
- 디자인 세부 사항보다 사용자 행동과 접근 가능한 요소를 중심으로 테스트해야 한다.
